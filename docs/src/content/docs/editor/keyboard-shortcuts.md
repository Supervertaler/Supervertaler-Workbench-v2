## File operations

| Shortcut | Action |
|----------|--------|
| `Ctrl+O` | Open file |
| `Ctrl+S` | Save file |

## Navigation

| Shortcut | Action |
|----------|--------|
| `Enter` | Move to next segment |
| `Shift+Enter` | Move to previous segment |
| `Ctrl+G` | Go to segment number |

## Editing

| Shortcut | Action |
|----------|--------|
| `Ctrl+Enter` | Confirm segment and move to next |
| `Ctrl+Z` | Undo |
| `Ctrl+Y` | Redo |
| `Ctrl+C` | Copy |
| `Ctrl+V` | Paste |

## Translation

| Shortcut | Action |
|----------|--------|
| `Ctrl+T` | Translate current segment with AI |
| `Ctrl+Shift+T` | Batch translate all empty segments |
