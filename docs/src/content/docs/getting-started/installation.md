Supervertaler Workbench is a desktop application for Windows, macOS, and Linux.

## Download

Download the latest release from GitHub:

**[Download Supervertaler Workbench](https://github.com/Supervertaler/Supervertaler-Workbench-v2/releases/latest)**

| Platform | File |
|----------|------|
| Windows | `Supervertaler-Workbench_2.0.0_x64-setup.exe` |
| macOS | `Supervertaler-Workbench_2.0.0_aarch64.dmg` |
| Linux | `supervertaler-workbench_2.0.0_amd64.AppImage` |

## Install

### Windows

1. Download the `.exe` installer
2. Run the installer – Windows may show a SmartScreen warning since the app is not yet code-signed. Click **More info** → **Run anyway**
3. Follow the installation wizard
4. Launch Supervertaler Workbench from the Start menu

### macOS

1. Download the `.dmg` file
2. Open the DMG and drag Supervertaler Workbench to your Applications folder
3. On first launch, macOS may block the app. Go to **System Settings → Privacy & Security** and click **Open Anyway**

### Linux

1. Download the `.AppImage` file
2. Make it executable: `chmod +x supervertaler-workbench_*.AppImage`
3. Run it: `./supervertaler-workbench_*.AppImage`

## System requirements

| Requirement | Minimum |
|------------|---------|
| OS | Windows 10+, macOS 12+, or Linux (glibc 2.31+) |
| RAM | 4 GB |
| Disk space | ~100 MB |
